# 배포 순서 (실수 방지용)

---

## STEP 1 — 💻 내 컴터에서

**`git-setup.bat` 실행**

→ 코드가 GitHub에 올라감
→ DB 파일은 안 올라감 (정상, .gitignore로 막혀있음)

---

## STEP 2 — 🖥 서버에서

**① `git-pull-server.bat` 실행**

→ GitHub에서 최신 코드 받아옴 + 서버 자동 재시작

**② `node migrate-variants.js` 실행** (터미널에서)

```
cd D:\price-list-app
node migrate-variants.js
```

→ 서버 DB에 단가 반영 (기존 데이터 안 날아감)

**③ 서버 재시작** (migrate 실행 후 필수)

→ `서버중지.bat` 실행 후 `테스트서버실행.bat` 실행
→ 브라우저 F5 새로고침

---

## ⚠️ 절대 하면 안 되는 것

| 하면 안 됨 | 이유 |
|-----------|------|
| 서버에서 `git-setup.bat` 실행 | 서버 코드가 GitHub를 덮어씌움 |
| 내 컴터에서 `node migrate-variants.js` | 내 컴터 DB만 바뀜, 서버엔 반영 안 됨 |
