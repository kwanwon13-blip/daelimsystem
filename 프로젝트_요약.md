# 대림SM 단가표 시스템 - 프로젝트 요약

## 기술 스택
- **백엔드**: Node.js + Express (server.js)
- **프론트엔드**: Alpine.js v3 SPA (public/index.html 단일 파일, 약 3400줄)
- **데이터**: JSON 파일 (data/data.json)
- **서버**: Windows, D:\price-list-app, 192.168.0.133:3000
- **네트워크 공유**: \\192.168.0.133\dd (D드라이브)

## 주요 파일
- `server.js` - 백엔드 API 전체
- `public/index.html` - 프론트엔드 전체 (Alpine.js SPA)
- `data/data.json` - 모든 데이터 (사용자, 견적, 업체, 옵션 등)
- `openfolder.vbs` - 커스텀 프로토콜 핸들러 (폴더/파일 열기)

## 기능 목록
1. **견적 작성/관리** - 품목 추가, PDF/Excel 내보내기, 이메일 발송
2. **단가 관리** - 카테고리별 단가, 면적별/폭별 구간 단가
3. **옵션 관리** - 고정단가/면적당/규격별 옵션
4. **업체 관리** - 업체 정보, 연락처
5. **사용자 관리** - 로그인, 권한, admin/user 역할
6. **시안 검색** - D드라이브 전체 이미지 인덱싱, 검색, 년도 필터
7. **연락처** - 현장별 연락처 관리
8. **통계** - 견적 통계 대시보드
9. **공지사항** - 사내 공지
10. **설정** - SMTP, 명함 설정

## 시안 검색 특이사항
- 커스텀 프로토콜 `openfolder://` + VBS로 클라이언트 PC에서 폴더/파일 열기
- 서버 토큰 방식으로 한글 경로 URL 인코딩 문제 해결
- 네트워크 경로 변환 (D:\ → \\192.168.0.133\dd\)
- .ai 파일 자동 감지 및 열기 버튼

## 알려진 제약
- index.html이 3400줄 이상으로 매우 큼 (분리 필요)
- IMAP(네이버) 메일 삭제는 VBA에서 불가 (복사만 가능)
