업체별 단가 입력 UI — 3가지 비교

직접 클릭/입력해보고 가장 편한 방식을 선택하세요

① 가로 매트릭스 표

행=품목, 열=업체 — 엑셀처럼 한눈에 모든 업체 단가 비교·입력

추천
품목명 규격 단위 최저가

💡 숫자 입력 후 Tab으로 이동 | 초록색 = 해당 품목 최저가 업체

② 품목 클릭 → 업체별 단가 모달

품목을 클릭하면 팝업에서 업체별 단가를 한번에 입력

③ 업체 탭 전환

업체 탭을 클릭하면 해당 업체의 단가 목록으로 전환

품목명 규격 단위 단가 (원)
① 매트릭스 표
✅ 한눈에 전체 비교 가능
✅ 최저가 업체 자동 하이라이트
✅ 빠른 Tab 이동 입력
❌ 업체 많으면 가로 스크롤
② 품목 클릭 모달
✅ 화면이 깔끔함
✅ 품목별 집중 입력
❌ 품목이 많으면 클릭 많음
❌ 전체 비교 어려움
③ 업체 탭 전환
✅ 업체별 집중 입력
✅ 화면 단순함
❌ 업체 간 단가 비교 불가
❌ 탭 많아지면 불편
