# 시안 도우미 — UXP 플러그인

Adobe Illustrator 2026+ 의 **진짜 도킹 가능한 패널** 플러그인.
레이어/색상 패널처럼 일러스트 안에 들어가 있음.

## 기능

1. **시안 정보 입력** — 규격/상품명/옵션/수량/납품일 (검색 가능 사전)
2. **표준 텍스트 자동 삽입** — 시안 안에 표준 형식으로 텍스트 프레임 추가
3. **시안 저장** — 표준 파일명/폴더 자동 + 4가지 형식 (원본 .ai, CS6 .ai, JPG, PDF)
4. **+ 옵션 추가** — 다중 선택으로 상품명에 `+양면+사방타공` 추가
5. **사전 검색** — 타이핑하면 실시간 필터

## 설치 방법 — UDT (UXP Developer Tool) 로 로드

### 1단계: UDT 설치 (한 번만)

Adobe 공식 UXP Developer Tool 다운로드:
- https://developer.adobe.com/photoshop/uxp/2022/guides/devtool/installation/
- Creative Cloud 데스크탑 앱 → Marketplace 검색 "UXP Developer Tool" → 설치

### 2단계: 플러그인 로드 (디자이너 PC마다)

1. **Illustrator 2026 실행**
2. **UDT 실행**
3. **[Add Plugin]** 클릭
4. 이 폴더의 `manifest.json` 선택:
   ```
   C:\...\price-list-app\scripts\illustrator-uxp\시안도우미\manifest.json
   ```
5. UDT에서 **[Load]** 클릭 (▶️ 아이콘)
6. Illustrator 메뉴: `Window > Plugin Panels > 시안 도우미` 클릭
7. 패널이 등장 → 우측 도크 영역에 드래그해서 도킹

### 3단계: 영구 설치 (선택, .ccx 패키징)

UDT 에서 **[•••]** > **[Package]** 클릭 → `.ccx` 파일 생성

직원 PC에 .ccx 더블클릭 → Creative Cloud 데스크탑이 자동 설치
(이렇게 하면 일러스트 켤 때마다 자동 로드, UDT 불필요)

## 사용법

1. 일러스트에서 시안 작업
2. **시안 도우미 패널**에서:
   - 규격, 상품명, 옵션, 수량, 납품일 입력
   - 미리보기에서 표준 형식 확인
3. **[텍스트 삽입]** → 시안에 표준 텍스트 프레임 박힘
4. 발주 단계:
   - **[💾 시안 저장]** → 저장 form 등장
   - 건설사/현장/형식 선택
   - **[💾 한 번에 저장]** → 4가지 형식 자동 저장

## 트러블슈팅

### 패널이 안 뜨는 경우
- UDT에서 [Reload] 클릭
- 일러스트 재시작

### "executeScript is not a function" 오류
- Illustrator 26.x (2024 이후) 필요
- 구버전 일러스트는 .jsx 스크립트 버전 사용 (`scripts/illustrator/시안제목입력.jsx`)

### 저장 시 권한 오류
- D 드라이브 쓰기 권한 확인
- 디자이너 PC에서 D:\ 매핑 또는 네트워크 경로 매핑 확인
- main.js 의 `DESIGN_ROOT` 수정 가능

### 한글 폴더명 문제
- 한글 윈도우면 자동 동작
- 영문 윈도우면 시스템 로케일 한국어로 설정

## 개발자 메모

- UXP 환경에서 Illustrator 조작은 `app.executeScript()`로 ExtendScript 실행 (가장 안정적)
- 미래 개선: UXP 네이티브 API로 점진 이전 (textFrames.add() 등 직접 호출)
- 마스터 사전(KIND_DICT 등)은 main.js 상단에 하드코딩 — 추후 ERP API 연동 가능
