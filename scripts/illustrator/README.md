# 일러스트 자동화 스크립트 모음

두 개의 독립 스크립트로 구성:

| 스크립트 | 시점 | 역할 |
|---|---|---|
| **시안제목입력.jsx** | 시안 작업 중 | 표준 형식 텍스트(규격/상품명/수량/납품일)를 다이얼로그로 입력 → 시안에 텍스트 프레임 자동 삽입 |
| **시안저장.jsx** | 시안 완료 후 | 표준 파일명/폴더 자동 생성 + 원본/CS6/JPG/PDF 한 번에 저장 |

각각 별도로 사용. 디자이너 워크플로우:
1. 시안 작업 시작 → `시안제목입력.jsx` 실행 → 다이얼로그에서 규격/상품명 등 입력 → 시안에 표준 텍스트 박힘
2. 디자인 마무리
3. `시안저장.jsx` 실행 → 폴더/파일명 자동 + 4종 형식 저장

# 시안저장.jsx — Illustrator 자동 저장 스크립트

## 기능

1. **다이얼로그 1번 클릭으로 자동 저장**
   - 건설사/현장/종류/내용/버전 → 드롭다운 + 자동완성
   - 발주처 다중 체크박스 (공장/코리아/한진/라코스/현진/대풍/배너스토어/한양안전/세계로/풍아몰/이용전/대영/현대상사/건우/동방사/서진/kep/풍아)
2. **자동 폴더 생성**: `D:\★건설사\YYYY시안작업\현장\`
3. **표준 파일명 자동 생성**: `MMDD-건설사-현장-종류-내용-v1-발주(발주처).{ai,jpg}`
4. **발주처 다중 선택** (체크박스, [전체 ON/OFF] 버튼)
5. **저장 형식 토글** (선택한 발주처에 일괄 적용):
   - 원본 .ai (Illustrator CC 호환)
   - 발주용 .ai (Illustrator CS6 호환) — 글꼴 100% / PDF호환 / ICC / 압축 옵션 포함
   - 발주용 .pdf (Acrobat 7 / PDF 1.6) — 편집기능보존 / 축소판 / Acrobat 레이어
   - 발주용 .jpg
   - ERP 자동 등록
6. 발주처마다 다른 형식 필요한 경우 → 스크립트 두 번 실행 (1차 / 2차)

## 설치

### 방법 1: 디자이너 PC마다 직접 설치 (추천)

1. `시안저장.jsx` 파일을 디자이너 PC로 복사
2. Illustrator 메뉴: `File > Scripts > Other Script...` (`Ctrl+F12`)
3. 위 파일 선택 → 다이얼로그 뜨면 OK

### 방법 2: 영구 메뉴 등록

Illustrator 설치 폴더에 복사하면 `File > Scripts` 하위 메뉴에 영구 등록됨:
```
C:\Program Files\Adobe\Adobe Illustrator 2024\Presets\ko_KR\스크립트\
```
(Illustrator 버전마다 경로 다름. CS6은 `Adobe Illustrator CS6\Presets\ko_KR\스크립트\`)

복사 후 Illustrator 재시작 → `File > Scripts > 시안저장`으로 실행.

## 설정 변경

스크립트 상단 `CONFIG` 객체에서 수정:

```javascript
var CONFIG = {
  ERP_HOST: "192.168.0.133",         // ERP 서버 IP
  ERP_PORT: "3000",                   // ERP 포트
  DESIGNER_TOKEN: "designer-default-key-change-in-env",  // ⚠️ 보안: .env 와 동일하게
  DESIGN_ROOT: "D:\\",                // 시안 저장 루트
  YEAR: new Date().getFullYear() + "시안작업"  // 년도 폴더 이름
};
```

## ERP 서버 설정 (관리자)

`/price-list-app/.env` 파일에 디자이너 토큰 설정:

```
DESIGNER_TOKEN=대림에스엠_designer_2026_secret_xxxxx
```

스크립트의 `DESIGNER_TOKEN`도 동일하게 맞춰야 함.

## 사용 흐름

1. Illustrator에서 시안 작업 완료
2. `File > Scripts > 시안저장` 또는 `Ctrl+F12` 로 스크립트 실행
3. 다이얼로그에서:
   - 월일 (자동 입력, 수정 가능)
   - 건설사 선택 (또는 직접 입력)
   - 현장 선택 (선택한 건설사의 현장 자동 필터)
   - 종류 선택
   - 내용 입력 (자유 텍스트)
   - 버전 (v1, v2, ...)
   - 발주처 체크 (다중 가능)
4. **[한 번에 저장]** 클릭
5. 자동으로:
   - 폴더 생성 (없으면)
   - 원본 .ai (CC) 저장
   - 발주처별 .ai (CS6) + .jpg 저장
   - ERP에 등록

## 결과 예시

**기본 케이스** — 발주처 = 공장, 디폴트 형식 (원본 CC + CS6 + JPG):

`D:\★두산\2026시안작업\성수장미\` 폴더에 자동 생성:
```
0429-두산-성수장미-A형철판-실명제-v1.ai              (원본 CC)
0429-두산-성수장미-A형철판-실명제-v1-발주(공장).ai   (CS6)
0429-두산-성수장미-A형철판-실명제-v1-발주(공장).jpg  (JPG, 저화질)
```
→ **3개 파일** (시안당 표준)

**발주처 없음** — 시안만 작업 시:
- 원본 .ai만 저장 (suffix 없음)

**발주처 다른 곳에도 보낼 때**:
- 스크립트 다시 실행, 발주처 다른 거 선택 → 새 파일 추가 생성

규칙:
- **시안 1개 = 1발주처** (동시발주 없음)
- **여러 발주처 필요 시 스크립트 여러 번 실행**
- 파일명 발주처 표기는 **CS6 ai / JPG / PDF에 붙음**, 원본 CC는 suffix 없음

## 트러블슈팅

### "Socket을 열 수 없습니다"
- ERP 서버(192.168.0.133:3000) 접근 가능한지 확인
- 디자이너 PC에서 `http://192.168.0.133:3000` 접속 테스트
- 방화벽 차단 여부 확인

### "스크립트 오류: ..." 다이얼로그가 뜸
- 일러스트 버전 확인 (CC 2018 이상 권장)
- 권한 문제면 일러스트를 관리자로 실행

### 파일이 저장되었는데 ERP에 안 보임
- 시안검색 인덱스는 30분 주기 자동 갱신
- 즉시 보고 싶으면 ERP의 [재인덱싱] 버튼

### CS6 저장이 안 됨
- 파일 → 다른이름저장 → AI 형식 → 호환성 옵션이 없는 효과 사용 시 경고 발생
- 일러스트 효과(블렌드, 변수 등) 단순화 후 저장
- 또는 디자이너에게 호환성 가이드라인 별도 안내

# 시안제목입력.jsx — 표준 텍스트 도우미

## 기능

시안 작업 중에 제목/메타 텍스트를 표준 형식으로 깔끔하게 입력하는 다이얼로그.

## 다이얼로그 구성

```
┌─ 시안 정보 ───────────────────────────────────────────┐
│  규격:    [가로*세로 ▼]  [600] * [900]                │
│           (또는 [지름(파이)] / [자유입력])             │
│  상품명:  [A형철판  ▼] 또는 [직접입력]                │
│  옵션:    [반사실사/단면]                             │
│  수량:    [2] 개   ☐ 다중 디자인                      │
│           (체크 시: 각 [_]개씩 총 [_]개)              │
│  납품일:  [4] / [29]                                  │
└────────────────────────────────────────────────────────┘
┌─ 미리보기 ────────────────────────────────────────────┐
│  600*900 A형철판 반사실사/단면 - 2개                  │
│  납품: 4/29                                           │
└────────────────────────────────────────────────────────┘
☐ 현재 선택 텍스트 위치에 삽입   ☑ 가운데 정렬

           [텍스트 삽입]  [취소]
```

## 동작

[텍스트 삽입] 클릭 시:
1. 활성 대지(아트보드) 중앙 위쪽에 텍스트 프레임 2개 자동 생성:
   - 1줄: `{규격} {상품명} {옵션} - {수량}` (24pt)
   - 2줄: `납품: {M/D}` (18pt)
2. 가운데 정렬 (옵션)
3. 디자이너가 위치/스타일 조정 가능

## 출력 형식 (표준)

| 입력 | 결과 텍스트 |
|---|---|
| 규격 600\*900, 상품명 A형철판, 옵션 반사실사/단면, 수량 2개 | `600*900 A형철판 반사실사/단면 - 2개` |
| 규격 550파이, 상품명 덧방원형스티커, 수량 5개 | `550파이 덧방원형스티커 - 5개` |
| 규격 300\*400, 상품명 포맥스3T, 다중디자인 각 30개씩 9종 → 총 270 | `300*400 포맥스3T - 각 30개씩 총 270개` |

## 설치

### 방법 1 (추천): Startup Scripts 자동 실행

일러스트 켜질 때마다 palette 자동 등장. **디자이너 PC마다 1번만 세팅하면 끝**.

**경로 (관리자 권한 필요)**:
```
Windows 일러스트 CC 2024:
  C:\Program Files\Adobe\Adobe Illustrator 2024\Support Files\Required\Startup Scripts\

Windows 일러스트 CC 2023:
  C:\Program Files\Adobe\Adobe Illustrator 2023\Support Files\Required\Startup Scripts\

Windows 일러스트 CS6:
  C:\Program Files (x86)\Adobe\Adobe Illustrator CS6\Support Files\Required\Startup Scripts\
```

**절차**:
1. 위 폴더를 관리자 권한으로 열기
2. `시안제목입력.jsx` 파일 붙여넣기
3. 일러스트 재시작
4. 끝 — 일러스트 켤 때마다 palette 자동 등장

> 주의: 같은 폴더에 `시안저장.jsx`도 함께 두어야 [💾 시안 저장] 버튼이 동작함.

### 방법 2: 임시 실행 (1회용)

설치 없이 바로 써보고 싶을 때:
1. `시안제목입력.jsx`를 디자이너 PC 임의 위치에 복사
2. 일러스트에서 `File > Scripts > Other Script...` (`Ctrl+F12`)
3. 위 파일 선택 → palette 등장

## 단축키 권장

자주 쓰는 스크립트라 단축키 등록 권장:
- Edit > Keyboard Shortcuts > Other Scripts
- 시안제목입력 → `Ctrl+Shift+T` (Title)
- 시안저장 → `Ctrl+Shift+S` (Save) — 일반 저장과 충돌 주의
