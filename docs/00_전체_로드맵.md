## 예정 기능 — 출퇴근/연차 (별도 Phase)

> 상세 설계 → docs/기능설계/출퇴근_연차.md

| Phase | 내용 | 상태 |
|-------|------|------|
| **Phase 0** | **직원 마스터 관리** (employees[] 추가, 사용자관리 탭에 직원목록 섹션, CAPS 이름 매핑) | ⬜ 대기 |
| Phase 1 | 출퇴근 기록부 (자동분류, 상태표시, 관리자수정, 부서별규칙) | ⬜ 대기 |
| Phase 2 | 연차 관리 (엑셀이관, 현황화면, 추가/공제, 촉진통지서) | ⬜ 대기 |
| Phase 3 | 연차정산서 (퇴사 시 회계년도/입사일 기준 비교) | ⬜ 대기 |

> ⚠️ Phase 0 먼저 해야 Phase 1~3이 제대로 동작함
> - 현재 출퇴근/연차는 ERP users[] 기준 → CAPS 계정 없는 직원(예: 안소현) 누락
> - employees[] 분리 후 CAPS 이름 매핑으로 해결
