## 계산 규칙
```
정규 근무: 08:30 ~ 18:00 (중식 1시간 제외) = 8.5시간
초과 근무: 18:00 이후 퇴근 시간
지각: 출근 > 08:30
조퇴: 퇴근 < 18:00
```

## server.js — 계산 함수
```js
function calcWorkHours(inTime, outTime) {
  if (!inTime || !outTime) return { 근무: 0, 초과: 0 };
  const [ih, im] = inTime.split(':').map(Number);
  const [oh, om] = outTime.split(':').map(Number);
  const inMin = ih * 60 + im;
  const outMin = oh * 60 + om;
  const total = Math.max(0, outMin - Math.max(inMin, 510) - 60); // 08:30 기준, 중식 1시간
  const regular = Math.min(total, 510); // 최대 8.5시간 (510분)
  const overtime = Math.max(0, total - 510);
  return { 근무: Math.round(regular / 60 * 10) / 10, 초과: Math.round(overtime / 60 * 10) / 10 };
}
```
