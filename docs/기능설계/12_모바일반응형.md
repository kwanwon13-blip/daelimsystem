# 3-1. 모바일 반응형 UI

- **난이도**: 상 | **효과**: 상
- **참고**: Frappe HRMS Mobile App

## 수정 파일
| 파일 | 작업 |
|------|------|
| `public/index.html` | @media 쿼리 전면 추가, 레이아웃 재설계 |

## 핵심 변환
```
데스크탑 → 모바일
사이드바    → 하단 네비게이션 바 (5개 주요 탭)
테이블      → 카드 리스트
모달        → 풀스크린 시트
버튼        → 최소 44px 터치 영역
```

## 우선 모바일화할 기능 (핵심만)
1. 출퇴근 기록 확인
2. 결재 처리
3. 공지사항 확인
4. 견적 목록 조회

## CSS 기본 구조
```css
/* 768px 이하 모바일 */
@media (max-width: 768px) {
  .sidebar { display: none; }
  .bottom-nav { display: flex; position: fixed; bottom: 0; width: 100%; }
  .main-content { padding-bottom: 60px; }
  table { display: none; }
  .card-list { display: block; }
}
```
