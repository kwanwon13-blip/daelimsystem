# 2-3. 견적 워크플로우 (작성→검토→승인→발송→완료)

- **난이도**: 중 | **효과**: 상

## 수정 파일
| 파일 | 작업 |
|------|------|
| `server.js` | PUT /api/quotes/:id/status 엔드포인트 추가 |
| `public/index.html` | 견적 목록에 상태 배지 + 상태변경 버튼 추가 |

## 상태 흐름
```
draft(작성) → review(검토요청) → approved(승인) → sent(발송) → done(완료)
                     ↓
               rejected(반려) → draft (재작성)
```

## server.js — 상태변경 API
```js
const VALID_TRANSITIONS = {
  'draft':    ['review'],
  'review':   ['approved', 'rejected'],
  'rejected': ['draft'],
  'approved': ['sent'],
  'sent':     ['done'],
};

app.put('/api/quotes/:id/status', requireAuth, (req, res) => {
  const quote = db.sql.quotes.getById(req.params.id);
  const { status: newStatus } = req.body;
  const current = quote.status || 'draft';
  const allowed = VALID_TRANSITIONS[current] || [];

  if (!allowed.includes(newStatus))
    return res.status(400).json({ error: `'${current}'→'${newStatus}' 변경 불가`, 허용: allowed });

  // 승인은 admin만
  if (newStatus === 'approved' && req.session.role !== 'admin')
    return res.status(403).json({ error: '승인 권한 없음' });

  db.sql.quotes.update(req.params.id, { status: newStatus });
  auditLog(req.session.userId, `견적 ${newStatus}`, quote.quoteName);
  res.json({ ok: true, status: newStatus });
});
```

## 상태 배지 색상 (index.html)
```js
quoteStatusBadge(status) {
  return {
    draft:    'bg-gray-100 text-gray-600',
    review:   'bg-yellow-100 text-yellow-700',
    approved: 'bg-green-100 text-green-700',
    rejected: 'bg-red-100 text-red-700',
    sent:     'bg-blue-100 text-blue-700',
    done:     'bg-purple-100 text-purple-700',
  }[status] || 'bg-gray-100';
},
quoteStatusLabel(status) {
  return { draft:'작성', review:'검토중', approved:'승인', rejected:'반려', sent:'발송', done:'완료' }[status] || status;
}
```
