# 1-4. 결재 브라우저 알림

- **난이도**: 하 | **효과**: 상
- **참고**: Frappe HRMS Push Notification, click2approve

## 수정 파일
| 파일 | 작업 |
|------|------|
| `public/index.html` | Web Notification API + 폴링 로직 추가 |

## 구현 내용
- `Notification.requestPermission()` 로 브라우저 알림 허용
- 30초마다 폴링: GET /api/approvals?status=pending → 새 건 감지
- 새 결재 요청 시 브라우저 알림 팝업
- 알림 클릭 → 결재 탭으로 이동

## 프론트 코드 (index.html function app() 내부)
```js
// init() 내부에서 호출
async startNotificationPolling() {
  if (!('Notification' in window)) return;
  await Notification.requestPermission();

  let lastCount = 0;
  setInterval(async () => {
    const res = await fetch('/api/approvals?status=pending');
    const data = await res.json();
    const count = data.length || 0;
    if (count > lastCount && Notification.permission === 'granted') {
      new Notification('결재 요청', {
        body: `미처리 결재 ${count}건이 있습니다`,
        icon: '/favicon.ico'
      });
    }
    lastCount = count;
  }, 30000); // 30초
}
```

## 주의
- 결재 탭 이미 있으므로 서버 변경 없음
- 알림 클릭 핸들러: `notification.onclick = () => { this.activeTab = 'approval'; }`
