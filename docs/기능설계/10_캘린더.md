# 2-5. 캘린더/일정관리

- **난이도**: 중 | **효과**: 상
- **참고**: ERPNext Calendar, FullCalendar CDN

## 수정 파일
| 파일 | 작업 |
|------|------|
| `db.js` | `'일정관리': { events: [] }` 추가 |
| `server.js` | GET/POST /api/calendar/events 추가 |
| `public/index.html` | 새 탭 "캘린더" 추가 (월간/주간 뷰) |

## 일정 데이터 구조
```js
{
  id: `evt_${Date.now()}`,
  제목: '현장방문 - 김포공항',
  유형: '현장방문',   // 현장방문, 미팅, 납기, 개인일정
  날짜: '2026-04-10',
  종료일: '2026-04-10',
  작성자: userId,
  공개: true,          // false면 본인만
  메모: ''
}
```

## 구현 방법 선택
- **간단**: CSS Grid로 직접 구현 (외부 의존성 없음)
- **풍부**: FullCalendar CDN (`https://cdnjs.cloudflare.com/ajax/libs/fullcalendar/`)

## 연동
- 연차관리 데이터 → 캘린더에 자동 표시 (색상 구분)
- 대시보드(06_대시보드.md) 홈화면에 "이번 주 일정 3건" 위젯 연동
- 납기일 입력 시 → 캘린더 자동 등록 (견적서와 연동 가능)
