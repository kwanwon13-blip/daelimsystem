# 2-4. 결재 대리/위임

- **난이도**: 중 | **효과**: 상
- **참고**: ERPNext Leave Approver Delegation

## 수정 파일
| 파일 | 작업 |
|------|------|
| `db.js` | `'결재위임': { delegates: [] }` 추가 |
| `server.js` | 위임 설정 API + 결재 라우팅 로직 수정 |
| `public/index.html` | 사용자 설정에 위임 설정 UI 추가 |

## 위임 데이터 구조
```js
{
  id: `del_${Date.now()}`,
  위임자: 'user1',       // 원래 결재자
  대리인: 'user2',       // 위임받는 사람
  시작일: '2026-04-01',
  종료일: '2026-04-05',
  사유: '출장'
}
```

## server.js — 위임 설정 API
```js
app.post('/api/approval/delegate', requireAuth, (req, res) => {
  const del = db['결재위임'].load();
  del.delegates.push({ id: `del_${Date.now()}`, 위임자: req.session.userId, ...req.body });
  db['결재위임'].save(del);
  res.json({ ok: true });
});

// 결재 요청 시 위임 여부 확인
function getEffectiveApprover(userId) {
  const today = new Date().toISOString().slice(0, 10);
  const del = db['결재위임'].load().delegates.find(d =>
    d.위임자 === userId && d.시작일 <= today && d.종료일 >= today
  );
  return del ? del.대리인 : userId;
}
```

## 결재 표시
- 위임 결재 시 "대리결재 (위임자명)" 뱃지 표시
- 위임 기간 종료 후 자동 해제 (날짜 비교)
