# 대림에스엠 메모 (데스크탑 앱)

ERP 워크스페이스의 데스크탑 클라이언트. S메모 / Sticky Notes 처럼 동작.

## 주요 기능

- 🪟 **Frameless + Transparent** 윈도우
- 📌 **Always-on-top** (네이티브, OS 도움 없이)
- 🎨 **색상 7종** (페이지별 기억)
- 🔍 **투명도 4단계** (호버 시 자동 100%)
- ▭ **미니 모드** (본문 숨기고 헤더만)
- 🛎️ **시스템 트레이** + 우클릭 메뉴
- ⭐ **즐겨찾기 메모** — 자동 시작 시 자동 오픈
- 💾 **양방향 동기화** — ERP 워크스페이스와 같은 DB
- 🚀 **Windows 자동 시작** (옵션)

## 빌드 방법 (관리자용)

### 1. 의존성 설치
```bash
cd desktop-memo
npm install
```

### 2. 개발 모드 실행 (테스트)
```bash
npm start
```

### 3. 빌드 (포터블 .exe)
```bash
npm run build
```

→ `dist/대림에스엠메모-portable.exe` 생성됨 (~80MB)

### 4. 인스톨러 빌드 (선택)
```bash
npm run build:installer
```

→ `dist/대림에스엠메모 Setup-1.0.0.exe` 생성

## 직원 배포

1. `dist/대림에스엠메모-portable.exe` 를 사내 공유 폴더 또는 USB 로 배포
2. 직원이 받아서 더블클릭 → 실행
3. 처음 실행 시 ERP 로그인 화면 → 평소처럼 로그인
4. 트레이 아이콘 우클릭 → 메뉴 표시

## 트레이 메뉴

```
+ 새 메모 만들기
─────────────────
⭐ 즐겨찾기 ▶ (핀 한 메모 목록)
🕐 최근 메모 ▶ (최근 5개)
📋 전체 메모 목록...
─────────────────
⚙ 설정 ▶
   - 항상 위에 표시 (기본값)
   - Windows 시작 시 자동 실행
   - ERP 서버 주소 변경
   - 로그아웃
─────────────────
🌐 ERP 웹에서 열기
─────────────────
종료
```

## 메모창 헤더 버튼

```
📌 [제목입력]  자동저장  👁  🎨  ▭  📌  ×
                       투명  색  미니  항상위 닫기
```

- **👁 투명도**: 클릭마다 100% → 80% → 60% → 40% → 100% 순환
- **🎨 색상**: 7종 (노랑/핑크/파랑/초록/보라/화이트/다크)
- **▭ 미니**: 클릭/더블클릭 → 헤더만 보이는 미니 위젯
- **📌 항상 위**: 윈도우 항상 위에 고정 토글
- **× 닫기**: 트레이로 최소화 (Shift+클릭 = 진짜 종료)

## 데이터

- **메모 내용** → ERP 서버 DB (회사 데이터)
- **창 위치/크기/색상** → 로컬 PC `%APPDATA%\대림에스엠 메모\config.json`
- **로그인 세션** → Electron 의 영속 쿠키 저장소

## 보안

- 회사 LAN 내부에서만 동작 (192.168.0.133:3000)
- 외부 접속은 클라우드플레어 터널 URL 입력 (설정 → 서버 주소 변경)
- 로그아웃 시 모든 메모창 닫고 세션 초기화

## 트러블슈팅

### "ERP 서버에 연결 실패"
- 사내망 접속 상태 확인
- 트레이 → 설정 → ERP 서버 주소 변경

### 메모창이 안 뜸 / 깨짐
- 트레이 → 종료 → 다시 실행
- 안되면: 로그아웃 → 재로그인

### 자동 시작 안 됨
- 트레이 → 설정 → Windows 시작 시 자동 실행 체크 다시 확인
- Windows 작업 관리자 → 시작 프로그램 에서 확인

## 개발자 노트

- Main process: `main.js`
- Preload: `preload.js`
- Renderer pages: `renderer/list.html` (메모 목록)
- 메모창 본체: ERP 서버의 `public/workspace-memo.html` (URL 로 로드)
- 데스크탑 모드 감지: `?desktop=1` 쿼리 또는 `window.electronAPI` 존재
- IPC 채널: `memo:*` 네임스페이스
