## 🔐 보안 기반 (2026-04-10 완료)

| 항목 | 내용 |
|------|------|
| 비밀번호 해시 | SHA256 → **PBKDF2** (100,000회 반복, 랜덤 salt) |
| 자동 마이그레이션 | 기존 SHA256 계정 → 로그인 성공 시 자동 PBKDF2 전환 |
| 로그인 실패 제한 | **5회 실패 → 30분 잠금** (IP + 계정 ID 이중 추적) |
| 급여 PIN 재인증 | `POST /api/auth/salary-pin` → 30분짜리 salary_token 발급 |
| 급여 세션 분리 | 메인 로그인 세션과 완전히 별도 관리 |
| 급여 접근 로그 | PIN 성공/실패/열람 모두 서버 콘솔 + 메모리 로그 기록 |
| 급여 권한 | `salary_view` 퍼미션 없으면 admin도 차단 |

## 🔜 우선 추가 추천 기능 (ERP 수주 경쟁력 순)

### 1순위 — 급여 모듈 (핵심)
> 실사용 엑셀 양식(대림에스엠/대림컴퍼니) 완전 분석 반영 (2026-04-15)

| 기능 | 세부 내용 |
|------|-----------|
| 급여 기초 설정 | 직원별 기본급/고정연장/고정휴일/식대/차량유지비/팀장수당, 월 소정근로시간, 통상시급 자동계산 |
| 4대보험 요율 설정 | 연도별 국민연금/건강보험/장기요양/고용보험 요율·상한·하한 관리 화면 |
| 소득세 유형 | 근로소득 100%/80%/120%, 10%~90%감면(청년), 사업소득 3.3%, 기타소득 8.8% |
| 급여 계산 | 월별 자동계산, 과세/비과세 분리, 포괄임금제(고정연장·휴일) 반영 |
| 지급 항목 | 기본급/연장/야간/휴일/고정연장·휴일/식대/차량/팀장/상여/소급/연차수당 + 자유항목 3칸 |
| 공제 항목 | 4대보험/소득세·지방세/정산소득세/연말정산/분할납부/4월추가분/환급금이자/과태료+주차비 + 자유항목 3칸 |
| 급여명세서 PDF | 개인별 생성, 산출근거 자동 포함, 일괄 생성 |
| 이메일 발송 | 개인별/일괄 발송, 발급대장 자동 기록 |
| 발급대장 | 언제 누가 발급/발송했는지 이력 관리 |
| 연간 급여현황 | 직원별 월별 집계 조회 |
| 급여대장 엑셀 | exceljs 기반 다운로드 |
| 간이세액표 관리 | 연도별 데이터 등록/수정 화면 |
| 두 회사 분리 | 대림에스엠 / 대림컴퍼니 별도 관리 |

> **보안**: `requireSalaryAccess` 미들웨어 → 로그인 + salary_view 권한 + PIN 재인증 3중 차단  
> **암호화**: 계좌번호 AES-256-GCM 암호화 저장

### 2순위 — 근태 고도화
> 현재 출퇴근 기록은 있으나 계산 자동화가 부족

| 연차수당 환산 | 미사용 연차 → 연차수당 → 급여 반영 | ❌ 없음 |

## 💡 개발 순서 권장

```
보안 기반 완료 (✅)
    ↓
급여 모듈 (salary_view 권한 + PIN 재인증 위에 구현)
    ↓
근태 시간외 자동계산 + 급여 연계
    ↓
인사기록 고도화 (인사기록카드, 제증명)
    ↓
평가/교육 관리
    ↓
회계/예산 (선택 - 대형 공공기관 수주 시)
```

## 🔑 급여 모듈 보안 사용법 (개발 시 참고)

```javascript
// routes/salary.js 에서 이렇게 사용
const { requireSalaryAccess, logSalaryAccess } = require('../middleware/auth');

// 모든 급여 API에 requireSalaryAccess 적용
router.get('/salary/records', requireSalaryAccess, (req, res) => {
  logSalaryAccess(req.user.userId, 'VIEW', '급여 목록 조회');
  // ... 급여 데이터 반환
});

// admin이 급여 PIN 설정
// POST /api/auth/salary-pin/set  { targetUserId, pin }

// 사용자가 급여 탭 열기 전 PIN 입력
// POST /api/auth/salary-pin  { pin }  → salary_token 쿠키 발급
```
