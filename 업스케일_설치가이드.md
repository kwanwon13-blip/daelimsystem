# 이미지 업스케일 설치 가이드 (Real-ESRGAN NCNN Vulkan)

AI 이미지 생성 결과(1024×1024) 를 최대 4배(4096×4096) 까지 업스케일하는 기능을 붙이려면 서버 PC 에 Real-ESRGAN NCNN Vulkan 바이너리와 모델 파일을 깔아야 합니다.

**쉬운 길: Upscayl 설치본에서 모델을 그대로 복사**하면 됩니다. ERP 카탈로그가 Upscayl 파일명과 동일하게 맞춰져 있으므로 **rename 필요 없습니다**.

GPU (NVIDIA/AMD/Intel) 가 있으면 빠르게, 없어도 CPU 로 동작합니다.

---

## 1. 서버 PC 폴더 구조 (최종 목표)

`D:\price-list-app\tools\realesrgan\` 아래에:

```
D:\price-list-app\tools\realesrgan\
├── realesrgan-ncnn-vulkan.exe      ← 실행파일 (별도 다운로드)
└── models\                          ← Upscayl 에서 그대로 복사 (rename 불필요)
    ├── ultrasharp-4x.bin

```

## 2. 실행파일(exe) 다운로드

Upscayl 에는 `.exe` 단독이 없으므로 따로 받습니다.

공식 릴리즈: https://github.com/xinntao/Real-ESRGAN-ncnn-vulkan/releases

→ `realesrgan-ncnn-vulkan-*-windows.zip` 받아서 압축 풀고 **`realesrgan-ncnn-vulkan.exe` 하나만** `D:\price-list-app\tools\realesrgan\` 에 복사.

같이 들어있는 `vcomp140.dll` 도 복사 (없으면 Visual C++ 재배포 패키지 설치).

## 3. 모델 파일 복사 (Upscayl 에서)

Upscayl 이 설치된 PC (예: 관리자 PC `192.168.0.30`) 에서 PowerShell 실행:

```powershell
# Upscayl 모델 폴더 → Desktop 스테이징 폴더로 복사
$src = "C:\Program Files\Upscayl\resources\models"
$dst = "$env:USERPROFILE\Desktop\realesrgan-models"
New-Item -ItemType Directory -Force -Path $dst | Out-Null
Copy-Item -Path "$src\*.bin","$src\*.param" -Destination $dst -Force
Get-ChildItem $dst | Select-Object Name, @{N='MB';E={[math]::Round($_.Length/1MB,1)}}
```

→ Desktop 에 `realesrgan-models` 폴더가 생김. 이 폴더를 USB 또는 네트워크로 **서버 PC 의 `D:\price-list-app\tools\realesrgan\models\`** 에 통째로 옮겨넣으면 됩니다.

**네트워크 공유 (관리자 공유 가능 시)**:
```powershell
# 서버 PC 의 D 드라이브 관리자 공유 사용 (서버 PC 관리자 자격증명 필요)
Copy-Item -Path "$env:USERPROFILE\Desktop\realesrgan-models\*" `
          -Destination "\\192.168.0.133\d$\price-list-app\tools\realesrgan\models\" `
          -Force
```

## 6. 사용

AI 탭 → 이미지 생성 → 생성된 이미지 아래 **🔍 크게 만들기** 바에서:
- 배율 2배 또는 4배 선택
- 모델 드롭다운에서 원하는 모델 선택 (또는 ✨ 자동 추천)
- **↑ 업스케일** 클릭

결과는 원본과 같은 폴더(`data/workspace-images/`) 에 `ai_xxx_4x_ultrasharp-4x.png` 형태로 저장되고, 이미지가 자동으로 교체됩니다.

## 8. 트러블슈팅

**"Real-ESRGAN 이 서버에 설치되지 않았습니다"**
→ `D:\price-list-app\tools\realesrgan\realesrgan-ncnn-vulkan.exe` 가 실제 존재하는지 확인.

**"모델이 설치되지 않음"**
→ `D:\price-list-app\tools\realesrgan\models\` 아래에 `.bin` 과 `.param` 파일 쌍이 들어있는지 확인. 확장자 대소문자 구분.

**업스케일 시 매우 느림 (30초 이상)**
→ GPU 드라이버 확인. NVIDIA 는 최신 드라이버, AMD 는 Adrenalin 최신. CPU 만 쓰면 4096px 기준 30~60초 소요 가능.

**Vulkan 관련 오류**
→ Vulkan 런타임 설치: https://vulkan.lunarg.com/sdk/home#windows

**VRAM 부족**
→ `-t` 옵션으로 타일 크기 조정 필요할 수 있음 (고급 사용자 전용, 현재 라우터에 미노출).
