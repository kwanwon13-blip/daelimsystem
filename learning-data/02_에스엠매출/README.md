# ② 에스엠 매출

**대상**: 대림에스엠(주) → 이카운트 매출 등록

## 들어갈 자료

### data/ — 정답 데이터
- 이카운트 매출 export 또는 판매현황 xlsx
- 헤더: "회사명 : 대림에스엠(주)" 인 파일
- **2601~지금까지 데이터 동기화 필요**

### scans/ — 매출 원본
- 매출 등록 시 사용한 자료
- 거래처 발주서, 견적서, 이메일 첨부 자료
- 디자이너 작업 결과물 (시안 PNG/PPTX)

### mappings/
- 매출 행 ↔ 원본 자료 매핑 JSON

## 거래처 (TOP 매출처)

- ㈜퍼시스 (인테리어 공사 — 코오롱타워, TKG태광 등)
