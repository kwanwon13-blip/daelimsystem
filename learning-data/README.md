# 📊 4분할 학습 데이터 풀

OCR 자동입력 모듈이 분할별로 학습할 데이터 폴더. **분할별로 완전히 분리** — 헷갈림 방지.

```
learning-data/
├── 01_에스엠매입/      → 이카운트 등록용
│   ├── data/             → 정답 엑셀 (이카운트 export)
│   ├── scans/            → 매입 명세서 사진/PDF
│   └── mappings/         → 매핑 결과 JSON
│
├── 02_에스엠매출/      → 이카운트 등록용
│   ├── data/             → 정답 엑셀 (판매현황 export)
│   ├── scans/            → 매출 원본 (시안/발주서)
│   └── mappings/         → 매핑 결과 JSON
│
├── 03_컴퍼니매입/      → E2E 등록용
│   ├── data/             → 정답 엑셀 (컴퍼니-매입매출.xlsx 매입행)
│   ├── scans/            → 매입 명세서
│   └── mappings/         → 매핑 결과 JSON
│
├── 04_컴퍼니매출/      → E2E 등록용
│   ├── data/             → 정답 엑셀 (컴퍼니-매입매출.xlsx 매출행)
│   ├── scans/            → 매출 원본 (PPTX 시안)
│   └── mappings/         → 매핑 결과 JSON
│
├── 99_품목마스터/      → 양사 공통 (시스템 전체 참조)
│
└── _무관_나이스텍/     → 별도 회사, 자동화 대상 X (참고용)
```

## 분할별 자료 구분 룰

### ① 에스엠 매입 (01_에스엠매입/)
- **정답**: 에스엠 → 이카운트에 등록한 매입 데이터 (export 엑셀)
- **원본**: 거래처에서 받은 매입 명세서 사진/PDF
- **거래처**: 삼성라코스, 이상테크윈, 동명이엔지, 보성세이프 등 안전용품 매입처
- **품목**: 안전화, 안전조끼, 안전그네, 방진마스크, 핫팩 등

### ② 에스엠 매출 (02_에스엠매출/)
- **정답**: 에스엠이 발행한 매출 데이터 (이카운트 또는 판매현황 export)
- **원본**: 매출 등록 시 사용한 자료 (시안, 발주서, 견적서 등)
- **거래처**: 퍼시스, HDC, 현대산업개발, DL E&C, 요진건설 등
- **품목**: 안전용품 + 잡자재(철물/보양)

### ③ 컴퍼니 매입 (03_컴퍼니매입/)
- **정답**: 컴퍼니가 받은 매입 (컴퍼니-매입매출.xlsx 매입수량 컬럼 있는 행)
- **원본**: 매입처에서 받은 명세서 (이노사인, 이노텍, 한울상사 등)
- **거래처**: 이노사인, 이노텍, 한울상사, 다모아안전 등 인쇄 자재 매입처
- **품목**: 후렉스출력, 그레이플렉스, 이노폼, 솔벤시트 등

### ④ 컴퍼니 매출 (04_컴퍼니매출/)
- **정답**: 컴퍼니가 발행한 매출 (컴퍼니-매입매출.xlsx 매출수량 컬럼 있는 행)
- **원본**: 매출 등록 시 받은 발주서 (PPTX 시안)
- **거래처**: 자매법인 에스엠 + 라코스/이상테크/동명/보성/디자인포트 등
- **품목**: 인쇄/사인물 (스티커, 후렉스+프레임, 3t포맥스 등) — **결합 품명 자주**

## 사장님 작업 안내

1. **사장님이 export 한 데이터 → 해당 분할의 `data/` 폴더에 넣기**
2. **명세서 사진/PDF → 해당 분할의 `scans/` 폴더에 넣기**
3. 시스템이 폴더 경로 기반으로 자동 분류 (파일명 추측 X)
4. AI 가 매핑 시도 → `mappings/` 에 결과 저장
5. 사장님 검토 후 학습 풀에 누적
