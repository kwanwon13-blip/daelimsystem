# ① 에스엠 매입

**대상**: 대림에스엠(주) → 이카운트 매입 등록

## 들어갈 자료

### data/ — 정답 데이터 (이카운트 export)
- 이카운트에서 [매입관리 → 매입조회 → 엑셀 export] 한 파일
- 또는 매입자동화.db 기반 export
- **2601~지금까지 데이터 동기화 필요**

### scans/ — 원본 명세서 (사진/PDF)
- 거래처에서 받은 매입 명세서 (세금계산서/거래명세서/영수증)
- 파일명에 일자 + 거래처명 권장 (예: `260415_삼성라코스.jpg`)

### mappings/ — 매핑 결과 (자동 생성)
- AI 가 명세서 ↔ 매입 데이터 매핑한 JSON
