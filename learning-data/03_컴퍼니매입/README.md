# ③ 컴퍼니 매입

**대상**: 대림컴퍼니(주) → E2E 매입 등록

## 들어갈 자료

### data/ — 정답 데이터
- 컴퍼니-매입매출.xlsx 의 **매입수량** 컬럼이 있는 행
- 이노사인 2월/3월 원장
- 이노사인_4월_E2E업로드 등

### scans/ — 매입 명세서
- 매입처에서 받은 명세서 사진/PDF
- 사장님이 앞으로 사용하면서 누적

### mappings/
- AI 매핑 결과

## 품목 패턴

인쇄 자재 — 단일 품명 위주

- 솔벤시트, 무광코팅지
- 결합 거의 없음
