견적 입력 UI 개선 — 현재 문제점 확인

지금까지 발견된 문제들이에요. 빠진 게 있거나 우선순위가 다르면 알려주세요.

A

팝업 위에 팝업 — 답답함

견적 팝업 위에 항목 편집 팝업이 또 뜨면서, 과거단가 사이드바도 가리고 화면이 겹겹이 쌓임. 한 화면에서 다 처리하고 싶음.

B

품목명 검색 불일치

"3t포맥스" vs "포맥스3T" — 순서가 다르면 검색이 안 됨. 키워드 분리 검색 필요.

C

품명 + 규격 동시 검색 불가

"포맥스 600*900" 같이 규격까지 같이 검색해야 실무에서 의미가 있는데, 지금은 품명만 됨.

D

입력 공간이 너무 좁음

이카운트처럼 넓은 테이블형 입력이 아니라, 좁은 팝업 안에서 입력해야 해서 답답함. 실제 내용은 별로 없는데 공간만 차지.

E

기타 — 다른 문제가 더 있음

위에 없는 다른 불편한 점이 있으면 터미널에서 알려주세요.
