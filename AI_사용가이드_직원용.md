# 🤖 ERP AI 기능 사용 가이드 (직원용)

## 어디서 쓰나요?

좌측 메뉴 → **AI 챗** 클릭 → AI 탭 진입.

## 모드 3가지

### 1️⃣ 일반 챗 (기본)

그냥 입력창에 질문 → Enter.

예시:
- "단가표 견적서 양식 어떻게 만들어?"
- "거래처에 메일 보낼 때 정중한 인사말 추천해줘"
- "엑셀에서 VLOOKUP 어떻게 써?"

**Shift+Enter** = 줄바꿈, **Enter** = 전송

### 2️⃣ 🎨 이미지 생성

🎨 버튼 클릭 → **품질 선택**:
- **초안** (~₩8) — 빠른 아이디어 스케치 / 컨셉 확인
- **시안** (~₩70) — ✅ 보통 이걸 사용. 클라이언트에 보낼 시안 품질
- **최종** (~₩280) — 인쇄/납품용 고화질

입력 예시:
- "현대적인 사무실 간판 디자인, 회색 배경"
- "협찬 행사 현수막, 빨강+노랑 컬러"

**참고 이미지 편집**: 이미지 첨부 후 🎨 모드 → "배경을 바다로 바꿔줘" 같이 입력 → 편집된 이미지 생성

생성 후 **🔍 크게 만들기** 바에서 4배 업스케일 가능 (4096×4096, 무료).

### 3️⃣ 🤖 Agent — 자동 작업

🤖 버튼 클릭 → 파일 첨부 + 작업 설명. AI 가 격리된 작업공간에서 알아서 처리.

**가능한 작업**:
- 엑셀 파일 정리/분석/차트
- 여러 파일 한 번에 합치기
- PDF 요약
- PPT 보고서 작성
- Word 문서 만들기
- ERP 데이터 가공

**예시**:
- "이 엑셀 거래처별로 정렬하고 합계 추가해줘" + (엑셀 파일 첨부)
- "지난 분기 매출 차트 만들어줘"
- "이 견적서들 비교해서 가장 싼 업체 표로 정리"
- "이 PDF 한 페이지로 요약해줘" + (PDF 첨부)
- "이 데이터로 PPT 보고서 5장 만들어줘"

**작업 시간**: 보통 30초 ~ 5분. 진행 상황 실시간 표시.

**결과 파일**: 작업 끝나면 다운로드 버튼 표시. 24시간 내 다운로드 가능.

---

## 📂 파일 첨부 방법

3가지 방법 다 가능:
1. **드래그 앤 드롭** — 입력창에 끌어다 놓기
2. **복사 붙여넣기** — Ctrl+C 한 이미지를 입력창에 Ctrl+V
3. **클립 버튼** (📎) 클릭 → 파일 선택

지원 형식: 이미지(jpg, png, webp), PDF, Excel(xlsx, csv), Word(docx), 텍스트

---

## ⚙️ 일일 사용 한도

화면 우상단에 표시됨 (예: `오늘 5/100`).

**일반 직원**:
- 챗: 100건/일
- 이미지 생성: 20건/일
- Agent 작업: 100건/일 (챗 한도와 공유)

**관리자**: 5배 한도

남은 횟수가 30회 미만이면 노란색, 10회 미만이면 빨간색으로 표시.

## 💡 잘 쓰는 팁

### 챗 모드
- **구체적으로** 물어보세요. "엑셀 도와줘" ❌  →  "엑셀에서 A열의 중복값을 빨강으로 표시하는 방법" ✅
- **회사 정보 포함**: "대림에스엠 단가표 양식" 처럼 구체적으로

### 이미지 모드
- **품질 vs 속도** 결정: 초안→시안→최종 단계 활용
- **참고 이미지 활용**: 비슷한 톤/구도가 있으면 첨부해서 "이런 느낌으로"
- **반복 시도**: 한 번에 안 나오면 프롬프트 수정해서 재시도 (저렴한 초안으로 빠르게 iteration)

### Agent 모드
- **파일은 반드시 첨부** — Agent 가 작업할 원본 데이터 있어야 정확
- **결과 형식 명시**: "엑셀로 정리" / "PDF 보고서" / "PPT 5장" 처럼 구체적으로
- **단계 분리**: 너무 복잡하면 작업을 단계별로 나눠서 요청
- **시간 여유**: 복잡한 작업은 3~5분 걸릴 수 있음, 다른 일 하면서 기다림

## ❓ 자주 묻는 질문

**Q: 작업 결과가 어디 저장되나요?**
A: AI 챗 대화 기록에 자동 저장. 하단 "히스토리" 에서 다시 열어볼 수 있음. Agent 결과 파일은 24시간 내 다운로드 가능.

**Q: 이미지 한 번 만들면 비용 차감되나요?**
A: 회사 계정 비용. 직원 개인 부담 없음. 다만 일일 한도 안에서 사용.

**Q: Agent 가 잘못된 결과를 만들면?**
A: 다시 요청 (프롬프트 다듬어서). 안 되면 일반 챗 모드로 단계별 도움 받기.

**Q: 나만 보는 대화? 팀과 공유?**
A: 기본은 본인만. 프로젝트(📁)로 묶으면 팀과 공유 가능.

## 🆘 문제 발생 시

- 답변이 안 옴 → 1분 기다린 후 새로고침
- "한도 초과" 에러 → 내일 다시 (또는 관리자에게 요청)
- 결과가 이상함 → 입력을 더 구체적으로 다시
- 그 외 → 관리자 (남관원)에게 화면 캡처 + 사용한 입력 공유
