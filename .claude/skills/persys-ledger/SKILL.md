# 퍼시스 자료 정리 절차

퍼시스 관련 요청은 사용자가 "엑셀로"라고 말하지 않아도 결과물을 엑셀 파일로 만들어야 한다.

## 기본 처리

1. 첨부 파일의 모든 시트를 확인하고 거래일자, 현장명, 품명, 규격, 수량, 단가, 공급가액, 부가세, 합계, 비고를 추출한다.
2. 같은 현장/같은 품목/같은 규격이 반복되면 합산 가능 여부를 판단하되, 원본 행 추적이 필요하면 원본행 번호를 비고에 남긴다.
3. 금액 검산은 `수량 * 단가`, `공급가액 + 부가세 = 합계` 기준으로 확인한다.
4. 불명확한 행은 버리지 말고 `확인필요` 시트에 남긴다.

## 출력 시트

- `정리본`: 청구나 공유에 바로 쓸 수 있는 표.
- `요약`: 현장별/품목별 합계.
- `확인필요`: 날짜, 품명, 규격, 금액 등이 불확실한 행.

## 응답

긴 표를 채팅에 붙이지 말고 `create_excel` 도구를 호출한다. 텍스트 답변은 파일 생성 완료와 확인필요 건수 정도만 짧게 말한다.
