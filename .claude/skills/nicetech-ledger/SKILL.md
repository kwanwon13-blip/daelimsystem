# 나이스텍 자료 정리 절차

나이스텍 관련 정리 요청은 결과물을 엑셀 파일로 생성한다.

## 기본 처리

1. 거래일자, 현장명, 품명, 규격, 수량, 단가, 공급가액, 부가세, 합계, 비고를 표준 컬럼으로 정리한다.
2. 현장명이나 품목명이 원본에서 여러 방식으로 적힌 경우 같은 항목끼리 묶되, 확실하지 않으면 `확인필요`로 분리한다.
3. 금액 계산과 합계를 검산한다.

## 출력 시트

- `정리본`
- `요약`
- `확인필요`

## 응답

`create_excel` 도구를 호출해서 다운로드 가능한 엑셀 파일을 만든다.
