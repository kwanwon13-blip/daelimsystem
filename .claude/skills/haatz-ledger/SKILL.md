# 하츠 자료 정리 절차

하츠 관련 요청은 거래처별 마감/청구용 자료로 보고, 기본 결과물을 엑셀 파일로 만든다.

## 기본 처리

1. 첨부 자료에서 거래일자, 거래처/현장, 품명, 모델명 또는 규격, 수량, 단가, 공급가액, 부가세, 합계를 정리한다.
2. 하츠 자료는 모델명/규격 표기가 중요하므로 원본 표기가 애매하면 임의로 바꾸지 않고 비고에 남긴다.
3. 중복으로 보이는 행은 합산 근거가 분명할 때만 합산한다.
4. 금액 검산이 맞지 않는 행은 `확인필요` 시트에 분리한다.

## 출력 시트

- `정리본`: 거래처 제출 또는 내부 확인용 정리 표.
- `요약`: 월별/현장별/모델별 합계.
- `확인필요`: 원본이 불명확하거나 금액 검산이 필요한 행.

## 응답

채팅 본문에 원본 표 전체를 출력하지 말고 `create_excel` 도구를 호출한다. 생성 파일 카드가 화면에 뜨도록 해야 한다.
