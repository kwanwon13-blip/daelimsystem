카카오 톡클라우드 단톡방 사진 자동 다운로드 작업 재개.

## 배경
- 대림에스엠(주)-업무방 단톡방 사진 전체를 D:\카톡사진모음\ 에 내려받는 중
- 톡클라우드 URL: https://talkcloud.kakao.com/teamchat/0439ba29399ceac90b61fd1017b87d8e/photo
- 약 2만 장 예상, 최근 일자(2026.4.24, 124장)부터 역순 진행 중
- 아직 다 다운로드 안 됐으면 이어받기, 다 됐으면 완료 메시지 남기고 종료

## 작업 순서
1. `mcp__Claude_in_Chrome__tabs_context_mcp` 로 현재 탭 확인
2. 톡클라우드 탭이 없으면 새로 열어 위 URL 로 navigate
3. 페이지 로드 후 `find` 툴로 "일자 체크박스" 중 **아직 체크 안 된 가장 최근 일자** 찾기
   - 예: ref 받아서 aria-label 에 "20YY. M. D" 포함한 checkbox
   - 이미 다운로드된 일자는 사용자 PC `D:\카톡사진모음\` 에 저장된 파일 날짜로 추정 가능
4. 해당 체크박스 클릭 → 그 일자 전체 사진 선택됨
5. `find` 로 "저장하기 버튼 (다운로드 아이콘 상단)" 찾아 클릭
6. 파일 저장 방식 대화상자 뜨면 "개별 파일로 저장" 유지 + "확인" 버튼 클릭
7. 60-90초 대기 (다운로드 완료)
8. 체크 해제 (또는 페이지 새로고침)
9. 다음 일자로 반복
10. 한 세션당 최대 10-20개 일자 처리 후 종료 (서버 부담 최소화, 1시간 뒤 다음 스케줄에서 이어받음)

## 주의사항
- 한 번에 한 일자씩만 선택 (1000장 제한 우려)
- 각 다운로드 사이 60초 이상 대기
- 크롬 다운로드 경로는 이미 D:\카톡사진모음\ 로 설정됨
- 브라우저 tier=read 이므로 컴퓨터-use 대신 Chrome MCP(mcp__Claude_in_Chrome__*) 사용
- 저장하기 버튼은 tier=read 브라우저 밖에서 클릭 안 되므로 Chrome MCP 의 left_click ref 방식 사용

## 완료 판정
- 스크롤 끝까지 내렸는데 체크 안 된 일자 없으면 완료
- 완료 메시지 남기고 종료

기존 세션이 끊겼을 수 있으니 스스로 모든 단계 수행할 것.
