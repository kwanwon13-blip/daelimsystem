# E2E 업로드 변환 스킬

## 개요

대림에스엠(주) ERP에서 내보낸 판매현황 xlsx 파일을 E2E 회계 시스템 업로드 형식으로 변환한다.

---

## E2E 업로드 컬럼 규칙

### 컬럼 순서 (A열부터 시작, 빈 열 없음)

| 열 | 헤더 | 원본 매핑 |
|----|------|-----------|
| A | 일자 | 일자-No. 에서 날짜만 추출 (YYYY-MM-DD) |
| B | 상품분류 | 공백 |
| C | 상품명 | 품목명 |
| D | 규격 | 규격 |
| E | 수량 | 수량 |
| F | 단가 | 단가 (소수점 보정 적용) |
| G | 금액 | 공급가액 |
| H | 세액 | 부가세 |
| I | 합계금액 | 합계 |
| J | 비고 | 비고 |
| K | 약어 | 공백 |

**중요: A열부터 바로 시작. 앞에 빈 열 절대 추가 금지.**

---

## 데이터 처리 규칙

### 포함 행
- 판매현황 시트 3행부터
- 일자 컬럼에 `YYYY/MM/DD` 형식의 날짜가 있는 행만

### 제외 행
- 월계, 총합계, 빈 행
- 일자 패턴 없는 행 (합계 집계 행)

### 날짜 변환
```
"2026/03/01 -244"  →  "2026-03-01"
```

### 소수점 단가 자동 보정
수량 × 단가 ≠ 금액인 경우 실제 단가를 역산하여 소수점으로 보정:
```python
if qty and price and amt and abs(qty * price - amt) > 0.01:
    price = round(amt / qty, 4)
```

---

## 검증

저장 전 전체 행에 대해 `수량 × 단가 = 금액` 검증 통과 확인 필수.

---

## 출력 파일명

```
{거래처명}_{월}월_E2E업로드.xlsx
예: 대림컴퍼니_3월_E2E업로드.xlsx
```

---

## 스타일

- 헤더: 맑은 고딕 Bold 10pt, 흰 글씨, 파란 배경 (#4472C4)
- 데이터: 맑은 고딕 10pt, 전체 테두리
- 숫자 컬럼 (수량, 단가, 금액, 세액, 합계금액): 오른쪽 정렬, `#,##0` 포맷
  - 단가가 소수이면 `#,##0.####` 포맷
- 일자: 가운데 정렬
- 틀 고정: A2 (1행 헤더 고정)

## 스크립트 실행

이 스킬 디렉토리의 `make_e2e.py` 사용:

