# 퍼시스 월별 마감내역서(거래명세서) 생성

퍼시스 관련 요청은 결과물을 **엑셀 파일**로 만든다. 원본을 못 읽으면 추정하지 말고 읽기 실패를 보고한다.

**방식: 전월 마감내역서(거래명세서) 양식을 그대로 복제한 뒤, 당월 `판매현황` 데이터로 안전/잡자재 2시트를 채운다.** (분석표/요약표를 새로 만드는 게 아니라, 거래처가 쓰는 마감내역서 양식 그대로 나와야 한다.)

## 실행 (이 스킬 폴더의 make_persys.py)

```bash
python "<이 스킬 폴더>/make_persys.py" --raw "<판매현황.xlsx>" --template "<전월 퍼시스-*.xlsx>" --outdir "<출력폴더>"
```

- `--template` 생략 시 raw 파일과 같은 폴더(및 outdir)에서 `퍼시스-*.xlsx` 최신본을 자동 탐색.
- `--outdir` 생략 시 raw 파일과 같은 폴더에 출력.
- 첨부 파일 경로를 `--raw` 에 넘기면 된다.

> Odysseus 메모: 이 SKILL.md 와 `make_persys.py` 는 같은 폴더(`data/skills/거래처마감/persys-ledger/`)에 있다. 에이전트는 그 폴더의 `make_persys.py` 를 파이썬으로 실행한다. (Odysseus 작업 경로에 맞춰 `<이 스킬 폴더>` 부분만 실제 절대경로로.)

## 템플릿(전월 양식)

- 처음 한 번은 **전월 마감내역서 xlsx 를 첨부**해야 양식이 등록·사용된다.
- 등록 후에는 **당월 판매현황 파일만** 올려도 같은 양식으로 자동 마감.

## 처리 내용

- 입력: 당월 `판매현황` 시트(거래처명/프로젝트명/품목/수량/단가/공급가액 등)
- 품목 구분: 비고 또는 품목명 규칙으로 `안전`/`잡자재` 2시트에 분배(`매출할인` 제외)
- 부가세가 비어 있고 공급가액이 있으면 10%로 계산
- 출력: **현장(프로젝트)별** `퍼시스-{현장명}-{MM월}.xlsx` (안전/잡자재 2시트)

## 종료 코드(실패 시)

`1`=판매현황 원본 없음 · `2`=전월 템플릿 없음 · `3`='판매현황' 시트 없음 · `4`=템플릿에 '안전/잡자재' 시트 없음 · `5`=데이터 0건 · `6`=결과 파일 0개

## 응답

긴 표를 채팅에 붙이지 말고 결과 엑셀 파일을 만든다. 답변은 파일 생성 완료·현장 수·행수 정도만 짧게.
